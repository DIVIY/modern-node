# Modern-Node

## Description

Modern Web Stack utilizing NodeJS as its central software development language. This software stack includes:

- Reverse Proxy - Traefik
- Database - Postgres
- Database Analytics - Metabase
- Cache - Redis
- Backend - NodeJS
- Frontend - React
- AB Testing - Sixpack
- Web Analytics - Matomo
- Monitoring - Netdata
